# ❤️ Healthcare Prediction System

## 📌 Overview

This project predicts the risk of heart disease using machine learning and provides insights through an interactive dashboard.

## 🚀 Features

* Data preprocessing & cleaning
* Machine learning model (Random Forest)
* Streamlit web app for prediction
* Power BI dashboard for visualization

## 🛠 Tech Stack

* Python (Pandas, Scikit-learn)
* Streamlit
* Power BI

## 📊 Results

* Achieved high accuracy using optimized ML model
* Identified key risk factors like age, cholesterol, and chest pain

## ▶️ How to Run

```bash
pip install -r requirements.txt
streamlit run app/app.py
```
